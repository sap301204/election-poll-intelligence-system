# 🇮🇳 India Election Poll Analytics Dashboard

🚀 A premium, AI-inspired election analytics dashboard built using Python and Streamlit.

This project simulates real-world election polling across India with interactive filters, advanced visualizations, and auto-generated insights — designed like a real analytics product.

---

## 🔥 Project Overview

Most dashboards just show charts.  
This system goes further — it helps **analyze, interpret, and understand voter behavior** across multiple dimensions.

👉 Built as an **end-to-end data product**:
- Data Generation → Cleaning → Analysis → Visualization → Insights

---

## 💡 Key Features

✔ Multi-state analysis (15+ Indian states)  
✔ Interactive filters (State, Region, Age, Gender, Candidate)  
✔ 🗺️ India state-wise bubble map  
✔ 📊 Candidate vote share & count analysis  
✔ 👥 Demographic breakdown (Age, Gender, Urban/Rural)  
✔ 📌 Issue-based voting insights (Education, Jobs, Inflation, etc.)  
✔ 📈 Polling trend over time  
✔ ⚡ Auto-generated insights panel  
✔ 🎨 Premium Neon UI Dashboard  

---

## 🧠 Tech Stack

- **Python** (Pandas, NumPy)
- **Streamlit** (Dashboard UI)
- **Plotly** (Interactive charts)
- **Matplotlib & Seaborn** (EDA)
